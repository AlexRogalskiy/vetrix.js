# Vetrix.js

> Converts a vector into a matrix

## Install

`$ npm install vetrix`

## Usage

### `vetrix(vector, columns)`

## Examples

```JS

var vetrix = require('vetrix');

var a = [1,2,3,4,5,6,7,8,9];
var b = vetrix(a, 3);
// b = [[1,2,3],[4,5,6],[7,8,9]]

```

```JS

var vetrix = require('vetrix');

var a = [1,2,3,4,5];
var b = vetrix(a, 2);
// b = [[1,2],[3,4],[5]]

```
